# tasker
A task manager for animation and vfx film projects.

requirement:
pyhton 2.7+
qtpy
sqlalchemy

and if you run it as a standalone tool and NOT from a dcc application like maya, nuke or houdini:
PyQt5
PyQt4
PySide2
PySide
